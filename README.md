# GitHub README Generator

A simple GitHub README Generator built with HTML, CSS, and JavaScript. It fetches GitHub profile data using the GitHub REST API and generates a professional README with a live preview.

---

## 🚀 Features

- Generate GitHub Profile README
- Live README Preview
- Dynamic Tech Stack Detection
- Dynamic Featured Projects
- Repository Description
- Copy README to Clipboard
- Download README.md
- Reset Form
- Theme Selection
- Responsive Design
- Error Handling

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GitHub REST API
- Fetch API
- Local Storage

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Deep-Kotak/GitHub-README-Generator.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

---

## 📖 How to Use

1. Enter your GitHub username.
2. Select a template.
3. Click **Generate README**.
4. View the live preview.
5. Copy or download the generated README.

---

## 🌐 GitHub API

This project uses the GitHub REST API.

User API

```
https://api.github.com/users/{username}
```

Repositories API

```
https://api.github.com/users/{username}/repos?sort=updated
```
